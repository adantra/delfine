\section{FEniCS/Dolfin}
\label{sc:dolfin}
De modo a automatizar a geração de código de baixo-nível do problema variacional necessário para a implementação dos métodos descritos no capítulo \ref{ch:form_num}, foi utilizada a ferramenta FEniCS/Dolfin, a qual consiste em uma série de chamadas (\emph{wrappers}) para rotinas otimizadas escritas em C++. Essa interface pode ser acessada tanto através da própria linguagem C++ quanto através do Python, tendo esta última alternativa sido escolhida devido a facilidade e rapidez para desenvolvimento de código com a mesma.
